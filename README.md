# claude-skills

A collection of Claude skills for Product Managers and teams.

## incident-comms

A Claude skill for Product Managers and Incident Communication Managers. Drafts ready-to-paste Slack incident announcements for non-technical stakeholders. Handles the full incident lifecycle: acknowledgement → assessment update → resolution → post-incident summary. Asks the right clarifying questions (status, data loss, affected products/services) before drafting. Can post directly to your incident announcements channel via Slack MCP when confirmed.

**Trigger phrases:** "incident", "outage", "service down", "write the incident message", "fix is deployed write the update", "draft the announcement"
